# Sestra Visuals Website

A clean, elegant, editorial portfolio website for Sestra Visuals branding studio.

## Tech Stack
- **Framework**: Next.js 14 (App Router)
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Language**: TypeScript

## Design Direction
- **Vibe**: Premium, Warm, Feminine, Intentional.
- **Typography**: Playfair Display (Serif) & Inter (Sans).
- **Colors**: Cream background, Deep Teal accents, Mahogany details.

## Deployment
This project is configured for seamless deployment on **Vercel**. Connect this repository to your Vercel account to enable automatic builds.

## Local Development
*Note: Ensure you have Node.js installed.*
```bash
npm install
npm run dev
```
